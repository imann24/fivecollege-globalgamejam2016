Five College Global Game Jam

Spring 2016

We are excited to host many aspiring and experienced game developers at the Hampshire site for the Global Game Jam 2016! Please complete this form to reserve your spot. We will start at 5pm on Friday, January 29th and jam for 48 hours. We hope you have lots of fun!

When: Starts at 5pm Friday, January 29th, 2016

What: 48 hours of game making fun (ends 5pm Sunday)

Why: To bring out the game developer in everyone!

Where: Adele Simmons Hall, Room 126, Hampshire College

Who: Anyone with an interest in making games, especially if you have some programming, art, audio, or game design skills.

To Participate, Please Do The Following:

  1. Register
    • Visit Global Game Jam
    • Click the "Join this Site" Button
    • Create an account if you haven't already registered
  2. (Optional) Like our Facebook Page and Spread the Word!
  3. (Optional) Tweet about the event! #fivecollegegjj
